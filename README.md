<h1>Boris Bikes</h1>

<h2>Introduction</h2>
This was the first project undertaken during the first week of Makers Academy.
The purpose of this was to develop a back end system in which users could pick up bikes from the docking station.
Also has breaking of the bikes functionality in which a van will collect all of the broken bikes, take them to the
garage to be fixed, and then return them to the docking station for use by users again.
All done using TDD (Test Driven Development), and using Object Oriented Design

<h2>Technologies used</h2>
 - Ruby
 - RSpec
